## Run the tests 

1. **Check PHP is available.** In a terminal opened *inside this folder*:
   ```
   php --version
   ```

2. **Download PHPUnit 10** as a single file from https://phpunit.de/ and save
   `phpunit.phar` into this folder (next to `phpunit.xml`).

3. **Run the suite** 
   ```
   php phpunit.phar
   ```
